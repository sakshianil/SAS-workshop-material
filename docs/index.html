Repository showcase

Clinical SAS Programming Workshops

The GitHub repository contains two structured tracks: a fully synthetic beginner workshop and a CDISC pilot mini-workshop using public reference artifacts.

Synthetic Phase III Workshop

A 10–12 hour hands-on path from CSV source data to SDTM-style domains, ADaM-style analysis datasets, patient listings, demographic summaries, AE reporting, figures, and QC checks.

CDISC Pilot Mini-Workshop

A shorter reference-based track for learners ready to inspect public CDISC pilot artifacts and understand traceability between DM, AE, ADSL, and ADAE.

Professional Habits

The materials emphasize logs, specifications, controlled terminology, derivation decisions, output manifests, validation checks, and reproducibility.

Learning pipeline

From source data to reportable evidence

  1. 01 Review protocol, SAP, source files, and mapping specs
  2. 02 Import and QC raw data in SAS OnDemand
  3. 03 Build SDTM-style domains such as DM, AE, EX, and VS
  4. 04 Derive ADaM-style analysis datasets such as ADSL and ADAE
  5. 05 Produce TLFs, compare QC outputs, and package evidence
