# Mock Statistical Analysis Plan

## General conventions

- Present treatment columns as Placebo, Active 100 mg, and Total.
- Analyze safety using the safety population.
- Age is derived at informed-consent/screening date. Partial birth dates use
  July 1 for year-only values and the 15th for year-month values.
- Study day is `date - treatment start + 1` on/after treatment start and
  `date - treatment start` before treatment start; there is no Day 0.

## Adverse events

A TEAE starts on or after first dose and no later than the subject's end date.
Count each subject once per summary level. Summaries include:

- Any TEAE.
- Serious TEAE.
- TEAE by severity.
- TEAE by system organ class and preferred term.

## Vital signs

Baseline is the scheduled baseline value. Change equals post-baseline result
minus baseline. Plot mean systolic blood-pressure change by visit and treatment.

## Outputs

- Listing 16.1.1: analysis populations.
- Table 14.1.1: demographics and baseline characteristics.
- Table 14.3.1: treatment-emergent adverse events.
- Figure 14.2.1: mean systolic blood-pressure change from baseline.
