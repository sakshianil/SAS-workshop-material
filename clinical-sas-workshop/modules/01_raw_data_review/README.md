# Module 01 — Raw Data Review

**Time:** 60 minutes  
**Work product:** `WORK.DATA_ISSUES`

Profile imported datasets with `CONTENTS`, `FREQ`, `MEANS`, SQL, and duplicate
checks. Identify partial dates, screen failures, inconsistent terminology,
ongoing AEs, missed doses, and early discontinuations.

**Workplace connection:** raw data is not “wrong” merely because it is messy.
Programmers document issues and apply agreed specifications without silently
inventing values.
