# Module 06 — ADAE

**Time:** 75 minutes  
**Work product:** `ADAM.ADAE`

Combine SDTM-style AE with ADSL. Derive numeric dates, relative study day,
treatment-emergent flag, treatment variables, severity number, and duration.
Review pre-treatment and ongoing events.

**Workplace connection:** a report should consume documented analysis
derivations rather than repeatedly re-derive them.
