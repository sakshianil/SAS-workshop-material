# Module 03 — SDTM-Style DM and AE

**Time:** 90 minutes  
**Work products:** `SDTM.DM`, `SDTM.AE`

Create one demographic record per subject and one adverse-event record per
event. Standardize identifiers, ISO dates, treatment labels, race, severity,
seriousness, preferred terms, and system organ classes.

**Important:** these are compact educational domains. Production mappings must
use the sponsor’s current licensed standards and controlled terminology.
