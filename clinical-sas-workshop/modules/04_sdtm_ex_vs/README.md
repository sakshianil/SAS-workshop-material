# Module 04 — SDTM-Style EX and VS

**Time:** 75 minutes  
**Work products:** `SDTM.EX`, `SDTM.VS`

Map repeated exposure and vital-sign records. Preserve missed-dose records,
standardize units and positions, assign sequence/order, and derive study days
relative to randomization.

**Workplace connection:** long-format domains make repeated observations
machine-readable and easier to validate.
