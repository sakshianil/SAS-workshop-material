# Module 00 — SAS Foundations in a Clinical Study

**Time:** 60 minutes  
**Work product:** `WORK.DM_SAMPLE`

Learn libraries, DATA steps, variable attributes, SAS dates, sorting, merging,
formats, procedures, macro variables, and ODS. The examples use real workshop
data so every syntax feature has a clinical purpose.

Run `lesson.sas`, attempt `exercise.sas`, then run `check_your_work.sas`.
Expected: five randomized subjects with numeric screening/randomization dates.

**Workplace connection:** clinical programs must be reproducible, readable in
the log, and explicit about variable type and length.
