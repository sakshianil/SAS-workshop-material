# Module 09 — AE Summary and Vital-Sign Figure

**Time:** 90 minutes  
**Outputs:** TEAE table, SOC/PT table, and systolic-BP change figure

Count subjects—not event records—for incidence summaries. Create baseline and
change-from-baseline vital-sign data, then plot mean change by visit and
treatment.

**Workplace connection:** denominator choice and analysis unit are statistical
requirements, not cosmetic report-programming decisions.
