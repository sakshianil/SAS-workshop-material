# Lab 01 — Submission Anatomy and XPT Import

**Time:** 45 minutes  
**Work product:** `WORK.METADATA_INVENTORY`

Learn how the selected artifacts relate:

- XPT carries SAS dataset records.
- Dataset-JSON describes dataset/column metadata in a machine-readable form.
- Define-XML describes study metadata and derivations across datasets.
- SDTM tabulations support data review; ADaM datasets support analysis.

Run the lesson after completing all setup programs. The exercise asks you to
inventory the four selected datasets and inspect the Dataset-JSON library
created by SAS's JSON engine.
