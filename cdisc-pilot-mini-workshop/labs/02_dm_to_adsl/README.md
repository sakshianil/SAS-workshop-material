# Lab 02 — Trace DM into ADSL

**Time:** 60 minutes  
**Work product:** `DERIVED.SUBJECT_LINEAGE`

DM contains 306 submitted demographic records; ADSL contains 254 analysis
subjects. Build a subject-level lineage view and inspect how treatment, dates,
age, sex, race, and population flags move into analysis.

The difference in record counts is itself meaningful: SDTM describes submitted
study observations, while an analysis dataset reflects defined populations and
analysis requirements.
