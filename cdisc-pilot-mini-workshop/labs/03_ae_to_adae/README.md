# Lab 03 — Trace AE into ADAE

**Time:** 60 minutes  
**Work product:** `DERIVED.EVENT_LINEAGE`

Join each SDTM AE record to its ADAE analysis record using `STUDYID`,
`USUBJID`, and `AESEQ`. Examine how ISO character dates become numeric analysis
dates and how ADSL subject attributes and analysis flags are added.

The reference ADAE program is available under `reference_programs`. Read it
after this lab and identify the corresponding derivations.
